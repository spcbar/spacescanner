/satellites get

get

returns a full list of satellites which will be really visible in the sky in the near future, with the exact time, for the GEO position provided!

get /satellites

returns a full list of satellites which will be really visible in the sky in the near future, with the exact time, for the GEO position provided!

  • Request
  • Response

Query Parameters

  • utc_time: required (datetime-only)

    current local time of the client in UTC

    Example:

    2019-02-28T22:26:26
  • lat: (number)

    current latitude of the client, optinal. If not provided, will be derived from IP address.

    Example:

    -39.86493451
  • long: (number)

    current longitude of the client, optional. If not provided, will be derived from IP address.

    Example:

    158.35261287
  • api_key: (string)

    API key from Spacescanner's internal data provider n2yo.com.

HTTP status code 200

Body

Media type: application/json

Type: object

[{
        "name": "SPACE STATION",
        "startUTC": 1521368025,
        "endUTC": 1521368660

HTTP status code 402

it means that Spacescanner does not have data available in the cache, so it is required to provide client's API key from n2yo.com. Please, repeat the request with api_key parameter provided.

Body

Media type: application/json

Type: object

Example:

{"error": "api_key is required for current GEO position"}
